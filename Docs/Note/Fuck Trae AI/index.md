# 傻逼Trae

我真受不了这个神人Trae了，写的代码如同狗啃的，诡异的代码补全占用了我的Tab键，而且它不是像VS Code那样的缩进优先，它是直接的整个占用，且拒绝后又会弹出来，很烦人！

并且它永远不会理解你的需求，毕竟用的是豆包的奶头乐模型，能听懂什么话？

>我让它写个让圆形的按钮，在鼠标悬浮时变成方形，按下后又变成圆形，结果它跟个傻逼一样给写成了：
>
>默认时：圆角矩形
>
>按下与悬停时：纯方
>
>然后我给改成了默认时纯圆，按下和悬浮圆角矩形，然后我改好后，它又给改成它写的狗屎了

AND

>我让它写个Python程序，结果它直接AI幻觉了一个依赖，然后就一直尝试安装这个依赖，即使这个依赖根本不存在，然后我也是直接重装系统了，我已没招

### 所以我再也不用这坨大粪了！
