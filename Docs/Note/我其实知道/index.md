# 我其实知道的
你在进入这个网站的时候，你也许会想——
### 这个网站有什么用？
### 何意味？
其实，这个网站，没有任何作用

我不需要让你知道这个网站的用处

因为这个网站，其实是为了满足我小时候的一个愿望——

我在很小的时候就希望我有一个网站，那时我还只会用Canva来弄个带许多超链接的项目，然后分享为网址。而现在长大了，我认识到了NodeJS、React、VitePress、Markdown……我就用VitePress搭配GitHubActions来搭建一个我小时候觉得搭建起来超难的网站。

而我，也知道这个网站没有用

但是，也正因如此，在这个网站，我也能继续当User

我不想把自己包装成一个黑客、技术大神……至少，我自己不喜欢。

因为我自己也是一名User，所以我不会用这个网站赚钱，也不会用这个网站去恶心其他的User，因为我觉得，这样对不起我自己

所以，我不可能把我的网站也变成那样

我不需要挣钱，也不需要恶心其他和我一样的User，因为我也是User
