# AiroTab

AiroTab is a clean and efficient new tab page application that provides search, theme, and wallpaper customization features. This project is built with modern frontend technology, has a responsive design, and can run well on different devices.

## Project Structure

The project contains the following main files:

1. `index.html` - Main page file, containing UI elements such as search box, settings icon, and settings panel
2. `script.js` - JavaScript file that handles user interaction, settings saving, and theme application
3. `styles.css` - CSS style file that implements a beautiful interface with Material You design style
4. `README.md` - Project description file (current file)

## Main Features

### 1. Search Function

- Provides a search box for users to input search content
- Supports multiple search engine options (Google, DuckDuckGo, Bing, Baidu, Sogou, 360 Search, Yandex, Naver)
- Allows setting search result opening method (current tab or new tab)

### 2. Settings Panel

- Accessible through the settings icon in the bottom right corner
- Contains three setting sections:
  - Search settings: Choose search engine and opening method
  - Appearance settings: Customize title style, clock style, theme color, and wallpaper
  - About: Display project information and version

### 3. Theme and Appearance Customization

- Provides multiple theme color options (purple, blue, green, red)
- Supports uploading custom wallpapers
- Can reset to default wallpaper
- Title/time display style switching

### 4. Responsive Design

- Adapts to different screen sizes (desktop, tablet, mobile)
- Mobile-friendly interface layout

## Technical Features

- Uses localStorage to save user settings
- Implements Material You design style
- Uses CSS variables to manage theme colors
- Adopts modern CSS features (Flexbox, Grid, etc.)
- Has good interactive feedback and animation effects
